# Chief

> Your agents don't need more power. They need a chief of staff.

Chief is the chief of staff for all your agents and information sources.
Everything flows into it; it thinks for itself; then it does one of three things:

1. 🔔 **Interrupt** the human — only when worth it, at the right moment, arriving with a plan.
2. 🤖 **Dispatch** work to downstream agents — and verify results before reporting.
3. 📚 **Curate** into memory — facts and intents waiting to be connected later.

## Quickstart

```bash
uvx --from . chief demo   # offline replay, zero keys, zero config
```
